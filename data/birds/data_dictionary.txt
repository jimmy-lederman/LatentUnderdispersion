Data Dictionary: Finnish Bird Abundance Data
=============================================

Source: https://github.com/braden-scherting/barcode

Files:
------

1. birds_final_counts.csv
   - Dimensions: 2826 rows (survey sites) x 137 columns (species)
   - Each column is a bird species (scientific name, e.g., "Anthus trivialis", "Fringilla coelebs")
   - Values: integer abundance counts observed at each site
   - First column: row index

2. birds_final_covariates.csv
   - Dimensions: 2826 rows (survey sites) x 21 columns (covariates)
   - All covariates are standardized
   - Variable descriptions can be found with the data source.


3. birds_final_names.csv
   - Dimensions: 137 rows (species) x 2 columns
   - Columns:
     - index: integer index for each species
     - species: scientific name of the species
